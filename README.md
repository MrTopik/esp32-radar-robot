#esp32-radar-robot

##Required Parts
1. ESP32
2. SG-90 Servo
3. HC-SR04 Ultrasonic Distance Sensor
4. A 3D Printer (I used an Ender 3 V3 SE and printed with PLA)
5. A soldering iron

##Assembly
1. Cut some jumper cables and solder only on ESP32
2. Get the cables out of the body from the top
3. Connect the servo and put it in (use some glue if needed)
4. Put the ultrasonic sensor in the head and connect the cables
5. Glue the head to the servo (You may have some problems you can use some other objects to help)
